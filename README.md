# Threat Model Dashboard

A simple web-based dashboard to document, classify, and mitigate security threats.  
Built with **HTML, CSS, and JavaScript**, this project provides a clean interface for listing threats, their severity, descriptions, and recommended mitigations. Users can also add new threats dynamically via a form.

---

## 🚀 Features
- Preloaded examples of common threats (SQL Injection, XSS, Weak Authentication).
- Card-based layout for readability.
- Add new threats dynamically using the form.
- Responsive design with clean styling.
